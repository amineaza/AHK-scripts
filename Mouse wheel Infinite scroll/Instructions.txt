1) Install autohotkey
2) Download the .ahk file
3) Run the file
4) You can put it in your windows startup folder for automatic execution
5) Enjoy!

Change mouse X1 button to X2 in the script to switch to the other lateral button

How to use:
Set your mouse lateral button to default if it was remapped to some other key
press the lateral button and scroll the wheel up or down, it will turn 100 more with each notch
you can release the lateral button
to stop the scrolling simply scroll up or down with the button released
